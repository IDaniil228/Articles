# Ежеденевник
Проект, в котором пользователь входит свою учетеную запись
В своем аккаунту заполняет список дел на каждый день недели.
В конце дня есть возможность отметить выполение заплонированных дел.
На основе этого будет выводиться статистика в процентах(на сколько пользователю удается придерживаться своего плана на день).

## Команды для запуска проекта
"python -m venv venv" - создание виртуального окружения
"venv/venv/Scripts/activate" - войти в виртульное окуружение
"pip install -r requirements.txt" - установка зависимостей
